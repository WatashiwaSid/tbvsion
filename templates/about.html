About TB VISION

Early Tuberculosis Prediction Using Deep Learning through CNN

TB VISION is my final year project aimed at developing an AI-powered system for early detection of tuberculosis from chest X-ray images. The project utilizes state-of-the-art deep learning techniques with Convolutional Neural Networks (CNN) to analyze chest X-rays and identify patterns associated with tuberculosis.

Tuberculosis remains a significant global health challenge, particularly in developing countries. Early detection is crucial for effective treatment and preventing the spread of the disease. This project addresses the need for accessible screening tools that can assist healthcare providers in identifying potential TB cases for further examination.

Technical Overview
  • Deep Learning Model: Custom CNN architecture trained on a large dataset of chest X-rays
  • Backend: Flask web framework with Python
  • Cloud Storage: Microsoft Azure Blob Storage for secure data management
  • Visualization: Custom heatmap generation for model interpretability
  • Reporting: Detailed PDF reports with visualization and analysis
Note: TBVISION is intended for educational and research purposes only. The predictions made by the tbvision neural network should not be used as the sole basis for clinical decisions. Please consult with a qualified healthcare professional for proper diagnosis and treatment.
